-------------

📄 server.js (Express Server)
├── Express app setup
├── Middleware configuration
│   ├── CORS

│   └── Static file serving

├── Error handling

📄 models/RepairRequest.js (Repair Request Schema)
├── Customer name (required)
├── Phone number (required, validated)
├── Email (optional)
├── Device model (required)
├── Issue description (required)
├── Images array
├── Status (pending/completed)
└── Created timestamp

📄 models/Accessory.js (Accessory Schema)
├── Name (required)
├── Description (required)
├── Price (required, number)
├── Image path (required)
├── Available (boolean)
└── Created timestamp

📄 models/Admin.js (Admin Schema)
├── Username (required, unique)
├── Password (required, hashed)
├── Password hashing pre-save hook
├── Password comparison method
└── Created timestamp

📄 routes/repairRequests.js (Repair Requests API)
├── POST /api/repair-requests
│   ├── Accept form data with images
│   ├── Validate input
│   ├── Upload images to storage
│   └── Save to database
├── GET /api/repair-requests (admin only)
│   ├── Fetch all requests
│   └── Return with image URLs
├── GET /api/repair-requests/:id (admin only)
│   └── Get single request details
├── PATCH /api/repair-requests/:id (admin only)
│   └── Update request status
└── DELETE /api/repair-requests/:id (admin only)
    └── Delete request and images

📄 routes/accessories.js (Accessories API)
├── GET /api/accessories (public)
│   ├── Fetch all accessories
│   └── Return with image URLs
├── POST /api/accessories (admin only)
│   ├── Accept form data with image
│   ├── Validate input
│   ├── Upload image
│   └── Save to database
├── PUT /api/accessories/:id (admin only)
│   ├── Update accessory
│   └── Replace image if provided
└── DELETE /api/accessories/:id (admin only)
    └── Delete accessory and image

📄 routes/admin.js (Admin Authentication API)
├── POST /api/admin/login
│   ├── Validate credentials
│   ├── Generate JWT token
│   └── Return token
└── POST /api/admin/create
    ├── Create admin user
    ├── Hash password
    └── Save to database

📄 middleware/auth.js (Authentication Middleware)
├── Extract JWT token from header
├── Verify token
├── Attach admin info to request
└── Handle authentication errors

API ENDPOINTS REFERENCE
========================

PUBLIC ENDPOINTS
----------------
GET    /api/accessories              # Get all accessories
POST   /api/repair-requests          # Submit repair request
GET    /api/health                   # Health check

ADMIN ENDPOINTS (Require Authentication)
-----------------------------------------
POST   /api/admin/login              # Admin login
POST   /api/admin/create             # Create admin (setup only)

GET    /api/repair-requests          # Get all requests
GET    /api/repair-requests/:id      # Get single request
PATCH  /api/repair-requests/:id      # Update request status
DELETE /api/repair-requests/:id      # Delete request

POST   /api/accessories              # Add accessory
PUT    /api/accessories/:id          # Update accessory
DELETE /api/accessories/:id          # Delete accessory
