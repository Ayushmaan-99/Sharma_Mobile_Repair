🆘 COMMON ISSUES & SOLUTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Issue: Backend not responding
Solution: Wait 60 seconds (free tier wakes from sleep)

Issue: Admin login fails
Solution: Check if admin user was created (run setup-admin.js)

Issue: Images not uploading
Solution: Check file size < 5MB, verify backend is running

Issue: CORS errors
Solution: Verify API_BASE_URL in js/config.js is correct

Issue: MongoDB connection fails
Solution: Check connection string, verify password is correct

🔄 HOW TO UPDATE YOUR WEBSITE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Make changes to files locally
2. Test changes locally
3. Commit and push to GitHub:
   
   git add .
   git commit -m "Description of changes"
   git push origin main

4. Automatic deployment:
   ├─ Vercel deploys frontend (2-3 minutes)
   └─ Render deploys backend (5-10 minutes)

5. Changes are live!
