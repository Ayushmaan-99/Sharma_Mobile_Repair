## 📐 Base Font Size Scaling

### Responsive Typography System
The entire website uses a **fluid typography system** that automatically scales based on device size:

| Device Size | Base Font Size | Effect |
|------------|----------------|---------|
| Desktop (>768px) | 16px | Full size |
| Tablet (≤768px) | 15px | 6% smaller |
| Mobile (≤480px) | 14px | 12% smaller |
| Small Mobile (≤360px) | 13px | 18% smaller |

**Benefits:**
- ✅ All text scales proportionally
- ✅ Maintains readability on small screens
- ✅ Reduces need for horizontal scrolling
- ✅ Better use of limited screen space

## 🎨 Visual Hierarchy on Mobile

### Priority Order
1. **Primary Actions**: Large, prominent buttons
2. **Key Content**: Headlines, important text
3. **Secondary Content**: Supporting information
4. **Tertiary Content**: Additional details
5. **Navigation**: Accessible but not intrusive

### Spacing System
- **XS**: 0.5rem (8px)
- **SM**: 0.75rem (12px)
- **MD**: 1rem (16px)
- **LG**: 1.5rem (24px)
- **XL**: 2rem (32px)

Mobile uses smaller spacing values for better space efficiency.

## 🔧 CSS Techniques Used

### Responsive Units
- **rem**: Relative to root font size (scales automatically)
- **em**: Relative to parent font size
- **%**: Percentage-based widths
- **vw/vh**: Viewport-relative units
- **fr**: Fractional units in Grid

### Modern CSS Features
- **CSS Grid**: Responsive layouts
- **Flexbox**: Flexible components
- **CSS Variables**: Consistent theming
- **Media Queries**: Breakpoint-based styling
- **Clamp()**: Fluid typography (future enhancement)

### Mobile-First Approach
- Base styles for mobile
- Enhanced for larger screens
- Progressive enhancement
- Graceful degradation

## 🆘 Troubleshooting

### Text Too Small
- Check base font size in html element
- Verify rem units are used
- Test on actual device (not just browser)

### Elements Overlapping
- Check padding and margins
- Verify z-index values
- Test in both orientations

### Horizontal Scrolling
- Check for fixed widths
- Verify max-width: 100%
- Test with long content

### Buttons Too Small
- Verify minimum 44px height
- Check padding values
- Test tap targets
